# TAQIKO（タキコ）

**TAQIKO**は、株式市場と競馬の両方で利益を追求するために開発された、二刀流の自動運用ツールです。

## 主な機能

1.  **株式売買戦略の検証・運用**
    * SNSやIRからデータを自動収集します。
    * TA-Libを用いて価格のテクニカル指標を算出し、株価データを分析します。
    * Qlibを用いて東証グロース市場での効果的な売買戦略を構築・検証します。

2.  **競馬の自動化**
    * 週末には、平日で得た利益を元手に、競馬の自動運用ツールとして機能します。
